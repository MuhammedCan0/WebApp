# Postman Tests für User-Auth-Tenant-service

Diese Datei beschreibt kurz, wie du die Postman-Collection nutzt.

## Voraussetzungen

- Postman installiert
- Services gestartet:
  - user-tenant-auth-service
  - gateway-service 

## Download und Import

1. Datei herunterladen: user-tenant-auth-service.postman_collection.json
2. In Postman: Import > File > die JSON-Datei auswaehlen


## Tests ausfuehren

1. Starte mit Ordner "0. Setup - Bootstrap Tokens".
2. Danach kannst du die Ordner 1 bis 7 der Reihe nach laufen lassen.
3. Du kannst auch einzelne Requests manuell starten und experimentieren.
